Где можно писать CSS:

  1. CSS как атрибут:

    2. пишем в элементах HTML с помощью атрибута style.

  2. CSS в head:

    3. используем тег style внутри head HTML-документа.

  3. CSS в отдельном файле:

    4. пишем CSS в отдельном файле с расширением .css, а затем связываем его с HTML с помощью тега link.

CSS состоит из трёх частей:

  • селектор - определяет, на какой элемент или элементы нацелиться;
  • свойство - определяет характеристики для изменения;
  • значение - определяет, как изменить эту характеристику.
